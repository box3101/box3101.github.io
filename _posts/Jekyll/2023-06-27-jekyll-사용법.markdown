## Jekyll 설치

#### 1. Jekyll 설치
```jsx
gem install jekyll
```  
#### 2. 필요한 gem들을 설치
```jsx
bundle install
```
#### 3. Jekyll 실행
```jsx
jekyll serve
```
#### 4. Jekyll build
```jsx
jekyll build
```
#### 5. git push
```jsx
git add .
git commit -m "메시지"
git push origin master
```
