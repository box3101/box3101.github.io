# 1. Cli 설치

```jsx
npm install -g @vue/cli
```

-**기본 vue 개발 환경을 설정해주는 도구**

-현재 vue 버전 확인

**vue --version**

```jsx
@vue/cli 5.0.8
```

# 2. 프로젝스 생성

**[뷰 프로젝트를 생성할 때 선택 옵션]**

```jsx
>vue init webpack-simple vue-todo
```

-작성

```jsx
Target directory exists. Continue? Yes
? Project name vue-todo
? Project description A vue.js project
? Author box4101@nate.com
? License Mit
? Use sass? No
```

-실행

```jsx
npm run dev
```
